# GameStart

Description

## Definition

```ruby
def GameStart
	int32 :command 
	int32 :unknown1  
	int32 :rec_owner  
	int32 :unknown3  
	int32 :unknown4  
	int32 :unknown5  
end
```

## Description

*:command*  
The command identifier for the action will always be `0x000001F4`.

*:rec_owner*  
The ID of the player that recorded the savegame.
